# animal_recognition

このアプリは、Streamlit を使用した動物認識アプリケーションです。ユーザーが動物園で撮った写真（jpg/png）をアップロードするか、カメラで撮影すると、生成AI（CLIPモデル）を使って画像に写っている動物を判定します。上位3つの予測結果と確率を円グラフで表示します。写真のファイル名付けに便利です。

## セットアップ

1. 必要なライブラリをインストール:
   ```
   pip install -r requirements.txt
   ```

2. アプリを実行:
   ```
   streamlit run app.py
   ```

## 特徴

- 生成AI（OpenAIのCLIP）を使用した動物認識。
- 動物園の一般的な動物（ライオン、ゾウなど）を対象。

- リアルタイム判定と視覚化。
