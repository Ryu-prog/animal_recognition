# ローカル環境での実行マニュアル

このドキュメントは、GitHubおよびStreamlit Cloudで公開している「動物認識アプリ」をローカルPC環境で動作させるための手順書です。

## 1. 事前準備

以下のファイルが同じフォルダ内に存在することを確認してください。

* 
`app.py`: アプリのメインUI制御プログラム 


* 
`model.py`: AIモデルの推論ロジック 


* 
`requirements.txt`: 必要なライブラリの一覧 



## 2. ライブラリのインストール

WindowsのPowerShellまたはコマンドプロンプトを開き、以下のコマンドを実行して必要なライブラリを一括インストールします。

```powershell
python -m pip install -r requirements.txt

```

インストールされる主なパッケージ:

* 
**streamlit**: Webインターフェース 


* 
**torch / torchvision**: AI計算エンジン（PyTorch） 


* 
**transformers**: CLIPモデルのロード 


* 
**Pillow / matplotlib / numpy**: 画像処理およびグラフ表示 



## 3. アプリの起動

インストール完了後、以下のコマンドでアプリを起動します。

```powershell
python -m streamlit run app.py

```

## 4. 初回起動時の操作

1. **Emailの入力**: 初回起動時のみターミナルにEmailを求められますが、空欄のまま **Enter** キーを押してスキップ可能です。
2. **ブラウザの起動**: 自動的にブラウザが立ち上がり、`http://localhost:8501` にアクセスされます。
3. **モデルのロード**: 初回の判定時にはAIモデル（CLIP）のダウンロードが行われるため、開始まで数十秒かかる場合があります。

## 5. 使い方

* 
**画像のソース選択**: サイドバーから「画像をアップロード」または「カメラで撮影」を選択します 。


* 
**一括判定**: 画像を複数（最大10枚）アップロードすると、一括で判定してZIPファイルで保存できます 。


* 
**結果の表示**: 判定された動物の名前、確率、および上位3件の割合を示す円グラフが表示されます 。
